# Bulgaria 2026 - Sunny Beach Vacation PWA

Responsive Progressive Web App for your Bulgaria trip (July 6-12, 2026).

## Features

- Full offline support (Service Worker)
- Daily itinerary with progress tracking
- Currency converter (ILS/EUR/USD to BGN)
- Packing checklist with progress bar
- Emergency contacts and hotel info
- RTL Hebrew interface
- "Add to Home Screen" support

## Run Locally

```bash
cd bulgaria-trip-2026
python server.py
```

Open http://localhost:8080 in your browser.

## Install as App

1. Open the site in Chrome/Edge on mobile
2. Tap "Add to Home Screen" or use the install button
3. The app works offline after installation
